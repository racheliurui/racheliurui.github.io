Posted on 2016-05-05 |

S1, have existing repo and change to use git repo to track the change

with remot repo
$ mkdir /path/to/new_repo
$ cd /path/to/new_repo
$ git --bare init

with existing repo
$ cd /path/to/existing_repo
$ git init
$ vi .gitignroe
$ git add .
$ git commit -m “init”

$ git push --set-upstream /path/to/new_repo mastergit
